The Spelunker

The Spelunker is a internal tool we built for poking around and investigating Who's On First data. It was so useful we made it public! The Spelunker lives at:

https://spelunker.whosonfirst.org/

To find out more about the Who's On First API we recommend reading the 2015 blog post introducing the Spelunker and Dan Phiffer's "WOF in a Box" series (parts one and two and three) about setting up and running the Spelunker locally.
