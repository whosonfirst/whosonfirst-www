Tools

Tools

Tools for Who's On First (WOF) are available from the whosonfirst GitHub organization.

Mapzen Places API

You can access WOF data via a REST API. Read the blog post or the documentation.

Web applications

  • Spelunker

    A tool for browsing the data. Check out our Spelunker blog post.

  • I Am Here

    A get lat lon style application for investigating specific points on the big planet-ball we all live on. Check out our I Am Here blog post.

  • Explore (work in progress)

    Discover what you can do with WOF records with this interactive tool.

  • Boundary Issues (work in progress)

    A web application for editing Who’s On First data. Check out the blog posts.

Desktop apps

  • electron-whosonfirst-api-explorer (work in progress)

    An Electron-based explorer for the Mapzen Places API.

  • electron-whosonfirst-bookmarks (work in progress)

    Bookmarks for Who's On First places.

Libraries

  • py-mapzen-whosonfirst

    A Python package to install the most common Who’s On First Python packages.

  • js-mapzen-whosonfirst

    JavaScript libraries for working with Who's On First data.
